Comfortably create Child Themes as well as quickly change the Theme to see what your website or publications with different Themes look like.

== Description ==
This plugin quickly generates a Child Theme from the top menu. It shows the list of available Themes of your Wordpress with the information of
each.
<br />
A child theme is a theme developed from a parent theme, which inherits its basic functionality. Although the child theme includes most of the features of its parent theme, it is a theme that does not update directly
<br />
Under each theme information there are 2 quick access buttons:

- **Custom Child**: Open the form for the creation of Child Theme.
- **Change / See**: Change the current theme for the selected one (this change is only reflected for the administrator for testing purposes).



== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Theme Switch X'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. You can see it in action in the Menu Bar

= Uploading in WordPress Dashboard =

1. Download `Theme-Switch-X.zip` from this page
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `Theme-Switch-X.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard
